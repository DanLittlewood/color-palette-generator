# How to Add a Screenshot

## Quick Instructions

1. **Open the app**: Open `index.html` in your browser
2. **Load an image**: Click "or load random photo"
3. **Pick colors**: Click on the image to generate a palette
4. **Take screenshot**:
   - **Mac**: Press `Cmd + Shift + 4`, then drag to select
   - **Windows**: Press `Win + Shift + S`, then drag to select
5. **Save as**: `screenshot.png` in this project folder
6. **Commit and push**:
   ```bash
   git add screenshot.png
   git commit -m "Add screenshot"
   git push
   ```

## What to Capture

Include in your screenshot:
- The main image with a color palette generated
- The left panel showing some palette history
- The right panel showing the live component preview
- One of the harmony modes active (Analogous/Complementary/Triadic)

## Ideal Window Size

- Width: ~1400px minimum
- Make sure all three panels are visible
- Browser zoom: 100%

The screenshot will automatically appear in the README once you add `screenshot.png` to the repo!
